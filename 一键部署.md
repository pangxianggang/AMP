1. **访问仓库设置**

3. **等待部署**

## ⚠️ 常见问题

### Q: 推送时提示认证失败？

**A**: 使用 Personal Access Token 代替密码
1. 访问：https://github.com/settings/tokens
2. Generate new token (classic)
3. 勾选 `repo` 权限
4. 复制 token
5. 推送时使用 token 作为密码

### Q: Pages 部署失败？

**A**: 检查以下几点
1. 确保选择了 `/website` 文件夹
2. 确保 `website/index.html` 存在
3. 查看 Actions 标签页的错误日志

### Q: 网站显示 404？

**A**: 
1. 等待 2-3 分钟
2. 清除浏览器缓存（Ctrl+Shift+R）
3. 检查文件夹是否选择正确
