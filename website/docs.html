AMP 文档

简介

AMP (Agent Memory Protocol) 是一个开放的记忆协议标准，让人工智能拥有长期记忆、从错误学习、持续成长的能力。

通过 AMP，您的 AI Agent 可以：

  • 记住用户偏好和历史对话
  • 从成功和失败中学习经验
  • 与其他 Agent 协作和知识共享
  • 随着使用时间不断成长

安装

# 从 PyPI 安装
pip install amp-protocol

# 或从源码安装
git clone https://github.com/pangxianggang/AMP.git
cd AMP/sdk/python
pip install -e .

快速开始

import amp
import asyncio

async def main():
    # 创建 Agent
    agent = amp.Agent(
        name="Ali",
        role="project_manager",
        language="zh"
    )
    
    # 存储记忆
    await agent.memory.remember(
        "用户喜欢中文回复",
        type=amp.MemoryType.SEMANTIC,
        importance=8
    )
    
    # 回忆记忆
    memories = await agent.memory.recall("用户偏好")
    print(f"记忆：{memories[0].content}")
    
    # 执行任务
    result = await agent.act({
        "description": "规划项目结构"
    })
    print(f"任务结果：{result}")

if __name__ == "__main__":
    asyncio.run(main())

Agent API

创建 Agent

agent = amp.Agent(
    name="Ali",
    role="project_manager",
    language="zh",
    storage_dir="~/.amp/agents"
)

记忆操作

# 存储记忆
await agent.memory.remember(
    content="用户偏好",
    type=MemoryType.SEMANTIC,
    importance=8,
    tags=["用户", "偏好"]
)

# 回忆记忆
memories = await agent.memory.recall("用户", limit=5)

# 记忆巩固
stats = await agent.memory.consolidate()

任务执行

result = await agent.act({
    "description": "完成任务",
    "importance": 8
})
